This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## Getting Started

The packet manager pnpm is used throughout the project.

To start the project on your own machine, pull it and perform the following commands :

```
pnpm install
pnpm dev
```
